rbenvでRubyの切り替えあるある

write date
2014年12月31日

概要

rbenvでRubyをインストールしてからrbenv global version_numberで切り替えるけど、適用されないあるある事例の自分用メモ。

$ ruby -v
ruby 2.0.0p451 (2014-02-24 revision 45167) [universal.x86_64-darwin13]

(´・ω・`)＜なんでやねん

$ vi ~/.bash_profile

#PATHを通すおまじない
export PATH = "$HOME/.rbenv/bin:$PATH"
eval "$(rbenv init -)"

:wq

#bashの設定ファイルを再読み込みするおまじない
$ source ~/.bash_profile

以上で、ruby -vで確認すると、

ruby 2.1.1p76 (2014-02-24 revision 45161) [x86_64-darwin13.0]

※Mavericksさんのデフォルトは2.0なので。
って感じに表示されればおｋ。
